* Eclipse: https://openjfx.io/openjfx-docs/#IDE-Eclipse

* javafx-controls 14-ea+7: https://mvnrepository.com/artifact/org.openjfx/javafx-controls/14-ea+7
* maven-compiler-plugin :https://mvnrepository.com/artifact/org.apache.maven.plugins/maven-compiler-plugin/3.8.1
* javafx-maven-plugin: https://github.com/openjfx/javafx-maven-plugin
